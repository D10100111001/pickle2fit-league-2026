# Team Logos

Please save your team logo images in this directory with the following names:

1. **naan-stop-picklers.png** - Naan-Stop Picklers team logo
2. **striking-falcons.png** - Striking Falcons team logo
3. **pickle-warriors.png** - Pickle Warriors team logo
4. **badmashers.png** - BadMashers team logo

## Image Requirements

- **Format**: PNG (recommended for transparency) or JPG
- **Size**: 400x400px or higher (square ratio recommended)
- **Background**: Transparent background works best

## How to Add Logos

1. Save each team logo image with the exact filename listed above
2. Place them in this `/public/logos/` directory
3. The app will automatically display them on:
   - Teams page (with team details)
   - Dashboard hero section (all 4 team logos)
